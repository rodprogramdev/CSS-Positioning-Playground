The Static position is the default position for all of the HTML elements. It is not positioned in any special way. It is always positioned according to the normal flow of the page.

The Relative position will be allowing the element to be positioned relative to its normal document flow, other elements will not be affected to fit into any gap left by the element.

The Fixed position will be relative to the viewport. This means it will always stay in the same place when the page is scrolled. We can use the top, right, bottom, and left properties to specify where we can position the element.

The Absolute position will be positioned relative to the ancestor or parent element. Unlike with the fixed position property that is relative to the view port.

The Sticky position will position the element based on the user scroll position.
